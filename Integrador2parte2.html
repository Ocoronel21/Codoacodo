Listas simples

Para crear una lista usaremos las siguientes etiquetas

  • Etiqueta ul: Coloca una lista desordenada
  • Etiqueta ol: Coloca una lista ordenada
  • Etiqueta li: Coloca un nuevo elemento en la lista

Para escribir una lista ordenada o desordenada haremos lo siguiente:

  1. Escribir las etiquetas ul / ol
  2. Dentro de ella escribir la etiqueta li
  3. Escribir el contenido dentro de cada etiqueta
